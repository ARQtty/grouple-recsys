Демо рекомендаций для GroupLe

Как это работает

По имени пользователя рекомендательная система находит его предпочтения, находит похожих на него пользователей. У них она находит объекты(мангу/додзинси/дораму), которые пользователь ещё не видел. Среди них выбираются те, которые должны больше всего понравиться пользователю

Как улучшить рекомендации

Дать системе больше данных. Сейчас рекомендации основаны на примерно 400000 пользователей (сейчас на портале их больше миллиона). Чем больше данных о предпочтениях пользователей будет знать система, тем лучше будут её рекомендации.

Сделать рекомендации ещё лучше можно, если добавить информацию о самом контенте: дать системе информацию о каждом объекте: жанр, количество страниц/серий, пример рисовки (пользователи часто отказываются читать мангу из-за неё).

Использовать персональные данные пользователей. Сейчас рекомендации основаны на публичном списке их закладок. У сайта есть данные о том, что пользователь читал и насколько долго. Имея эту информацию, система будет ещё лучше понимать предпочтения.
